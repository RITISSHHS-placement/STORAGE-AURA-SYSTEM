## 🚀 Deploy in 5 Minutes (GitHub → Render → Permanent URL)

### Step 1 — Push to GitHub

```bash
# Clone or download this project, then:
git init
git add .
git commit -m "Initial commit — AURA STORAGE"

# Create a new repo on github.com, then:
git remote add origin https://github.com/YOUR_USERNAME/aura-storage.git
git branch -M main
git push -u origin main
```

### Step 2 — Deploy on Render (free, permanent URL)

1. Go to **[render.com](https://render.com)** → Sign up / Log in
2. Click **"New +"** → **"Web Service"**
3. Connect your GitHub account → select **aura-storage** repo
4. Render auto-detects `render.yaml` — just click **"Deploy Web Service"**
5. Wait ~2 minutes → get your permanent URL: `https://aura-storage.onrender.com`

**That's it.** Every `git push` auto-redeploys. No CLI, no config, no URL to remember.
